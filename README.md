## Setup

- `npm install`
- Create `.env.local` with `DATABASE_URL`, `TWILIO_ACCOUNT_SID`, `TWILIO_AUTH_TOKEN`, `TWILIO_PHONE_NUMBER`, `OPENAI_API_KEY`, `NEXT_PUBLIC_APP_URL`
- `npm run db:generate && npm run db:migrate`
- `npm run dev`

## Stack

Next.js 15 + TypeScript + Supabase + Twilio + OpenAI
